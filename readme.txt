== Description ==

~English~
This plugin only deletes the bbp roles that bbpress create. It's useful after uninstall the bbpress plugin.
To use it you just have to activate it and immediately deactivate it. Then, you can delete this plugin.

~Spanish~
Este plugin tan solo elimina los roles que crea bbPress después de su desinstalación.
Para usarlo tan solo debes activarlo para acto seguido desactivarlo y borrar el plugin.

== Installation ==

1. Upload this folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Deactivate the plugin and delete it.

== Frequently Asked Questions ==

= What I have to do? =

Nothing but activate and deactivate the plugin.
